Completed System Industrial R&D / QA Tool

Automated Aerodynamic Test Stand (SBSC v.1.0)

A complete hardware-software solution for measuring pressure drops in ventilation components. The system features a custom USB controller, a desktop application for test management, and automated data processing with PDF/CSV report generation.

Project Goal: Automating Quality Assurance

The primary challenge was to replace manual, time-consuming, and error-prone testing methods with a reliable, repeatable, and computer-controlled system. The goal was to create a tool for the Quality Assurance department that could accurately characterize the aerodynamic performance of ventilation components (ducts, elbows, etc.) and automatically generate standardized test documentation.

My Role: System Architect & Developer

As the sole designer and developer, I was responsible for the entire system from concept to deployment. This involved:

  • Designing the custom hardware controller and its communication protocol.
  • Developing the desktop application in C# (.NET) to act as the Human-Machine Interface (HMI).
  • Implementing the data acquisition and processing logic.
  • Integrating the entire solution into a functional, production-ready test stand.

Technical Solution: A Hardware-Software Co-Design

The system is a classic example of hardware-software co-design, where each component was developed to work in synergy.

  1. Hardware Layer: A custom USB-based controller interfaces with the high-power ventilation fan and a pair of high-precision pressure sensors (0-2000Pa range) placed in the aerodynamic tunnel.
  2. Software Layer (Desktop App): The C#/.NET application provides a user-friendly interface to configure test parameters (part number, operator data), initiate the automated test sequence, and manage results.
  3. Automated Data Processing & Reporting:
    • The system automatically performs a sequence of 5 measurements, then applies a **statistical filtering algorithm** (discarding min/max values and averaging the rest) to ensure data robustness against outliers.
    • Results are stored in a local database to generate a pressure drop vs. fan speed characteristic curve.
    • Operators can instantly generate a formal **PDF report** with an automatic timestamp or **export raw data to CSV** for further analysis.
