Live Production System Pro Bono Engineering

Distributed Stage Lighting & Audio Control System

A distributed ESP32-based Master/Slave architecture integrating lighting (PWM, SSR, DMX) with multi-track audio playback via Web Audio API. Designed for full theatrical productions with script synchronization and mobile rehearsal control.

Developed pro publico bono for a local amateur theatre group.

System Architecture

The system consists of one MASTER controller and two SLAVE units communicating via ESP-NOW protocol.

  • 4x PWM dimmers (16A per channel, with RC snubbers & filtering)
  • 4x SSR outputs (40A ON/OFF reflectors)
  • Independent DMX output (MAX RS-485 – 512 channels per controller)

Total scalable capacity: 1500+ DMX channels.

Web-Based HMI (SPIFFS + HTML + JS)

  • Scene list with programmable time markers (hh:mm:ss)
  • Full blackout emergency button
  • Skip sequence control
  • Real-time communication monitoring
  • Remote reset of slave controllers
  • Embedded PDF script viewer

Mobile Control During Rehearsals

During stage rehearsals, the lighting system was controlled directly from a smartphone browser. The operator walked alongside the director, triggering scene sequences in real-time.

This significantly increased flexibility and reduced iteration time when tuning transitions according to the director's requirements.

Audio Engine (Web Audio API)

Integrated multi-track playback (up to 32 channels) compatible with Behringer X32 mixer via USB multi-channel interface.

  • Drag & drop playlist editing
  • Time-based navigation (hh:mm:ss jump)
  • Synchronization with script timestamps

Reliability & Safety Engineering

  • ESP-NOW low-latency communication (20m+ range)
  • MAC binding for slave authorization
  • RC snubber circuits for AC stability
  • Independent DMX universes per controller

Live Performances

The system was successfully used in full-scale live productions with hundreds of attendees.
