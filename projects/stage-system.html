Live Production System

Distributed Stage Lighting & Audio Control System

A distributed ESP32-based Master/Slave architecture integrating lighting (PWM, SSR, DMX) with multi-track audio playback via Web Audio API. Designed for full theatrical productions with script synchronization.

System Architecture

The system consists of one MASTER controller and two SLAVE units communicating via ESP-NOW protocol. Each controller includes:

  • 4x PWM dimmers (16A each, with RC snubbers & filtering)
  • 4x SSR outputs (40A ON/OFF high-load reflectors)
  • Independent DMX output (MAX RS-485) – 512 channels per controller

Total scalable capacity: 1500+ DMX channels.

Web-Based HMI (SPIFFS + HTML + JS)

The MASTER unit hosts a WiFi network ("OSWIETLENIE SCENICZNE") and serves a full web-based control interface.

  • Scene list with programmable time markers (hh:mm:ss)
  • Full blackout ("WYGAŚ WSZYSTKO") emergency button
  • Skip sequence control
  • Real-time communication status
  • Remote reset of slave controllers
  • Embedded PDF script viewer for operator synchronization

Mobile Control During Rehearsals

During stage rehearsals, the lighting system was controlled directly from a smartphone browser. The operator walked alongside the director, triggering scene sequences in real-time, allowing instant adjustments without being physically tied to the control console.

This significantly increased flexibility and reduced iteration time when tuning transitions and timing according to the director's vision.

Audio Engine (Web Audio API)

Integrated multi-track playback (up to 32 channels) using Web Audio API. Compatible with Behringer X32 via USB multi-channel interface.

  • Drag & drop playlist editing (even during playback)
  • Time-based navigation (manual hh:mm:ss jump)
  • Full synchronization with script timestamps

Reliability & Safety Engineering

  • ESP-NOW for stable low-latency communication (20m+ range)
  • MAC address binding to prevent unauthorized slave pairing
  • RC snubber circuits for AC noise suppression
  • Separate DMX universes per controller

Scalability

The system can be expanded by adding additional SLAVE units. Each new controller is registered in the MASTER firmware via MAC address mapping.

Live Performances

The system was successfully used in full-scale live productions with hundreds of attendees.
